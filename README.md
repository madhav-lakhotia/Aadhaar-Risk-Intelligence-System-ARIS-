## Overview
Aadhaar Risk Intelligence System (ARIS) is a data-driven analytics project that identifies patterns, anomalies, and risk indicators in Aadhaar enrolment and update data. It analyzes regional variations across districts and pincodes to support smarter and more efficient Aadhaar service delivery.

The system uses biometric behaviour (bio-age) vs demographic age (DOB) analysis to detect unusual patterns and highlight regions with higher update demand or irregular activity.

---

## Objective
- Identify unusual enrolment and update patterns  
- Analyze biometric age vs demographic age differences  
- Detect regional variations in Aadhaar usage  
- Generate insights for better resource planning  
- Enable data-driven decision making

## How It Works
1. Collect Aadhaar enrolment and update datasets  
2. Clean and preprocess data using Python  
3. Analyze patterns across regions and age groups  
4. Compare biometric age with demographic age  
5. Detect anomalies and unusual trends  
6. Generate insights for resource optimization
